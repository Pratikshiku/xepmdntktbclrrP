## Colorful Yunnan Cultural Tourism Website

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/78160ce944a5449bb351e3a42d51b595/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Feature Description
The system is divided into (roles): Regular users (Tourists), Tour guides, Administrators.

**Regular User Features:**
1. Login
2. Registration
3. Homepage
4. Forum
5. Tourist Attractions
6. Hotel Information
7. Message Board
8. Announcement Information
9. Personal Center
10. Tourist Attraction Favorites
11. Hotel Favorites
12. Hotel Booking


**Tour Guide Features:**
1. Login
2. Registration
3. Change Password
4. Personal Information
5. Tourist Attraction Management
6. Tourist Attraction Comment Management
7. Hotel Information Management
8. Hotel Comment Management
9. Forum Management
10. Announcement Information Management



**Administrator Features:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Tourist Management
5. Tour Guide Management
6. Tourist Attraction Management
7. Tourist Attraction Comment Management
8. Tourist Attraction Favorite Management
9. Hotel Information Management
10. Hotel Comment Management
11. Hotel Favorite Management
12. Hotel Booking Management
13. Message Board Management
14. Attraction Type Management
15. Room Type Management
16. Announcement Type Management
17. Forum Management
18. Announcement Information Management
19. Carousel Management
